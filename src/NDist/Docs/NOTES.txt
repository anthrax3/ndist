NDIST Features

- HOST
  + Host services in seperated application domains.
  + Remotely connect to servers, start/stop services.

- INSTALL/UNINSTALL
  - Install/uninstall a service remotely

- UPDATE
  - Manual check & update of a service (mass update on all servers)
  - Periodically check & update of configured services in a server.

- MESSAGE
  - Async & Sync messaging
  - Request/Repose style messaging
  - Reliable, Loose coupled, guarantied delivery, Send and Forget type messaging.
  - Messaging between services in same or different servers.
  - Remote method invocation on top of messaging

- DEPENDENCY
  - Services can have dependent services. So, when installing a service, all depended services are installed.
